# BlogApp

BlogApp is a REST application developed with Node.js, Express.js and MongoDB. Semantic UI CSS library was implemented for styling. 

## Installation

Use the package manager [npm](https://www.npmjs.com/package/npm) to install.

```bash
npm install
```

## Run Local

```javascript
npm run local
```
